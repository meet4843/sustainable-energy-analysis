# Sustainable Energy Consumption & Carbon Emission Analysis

## Project Overview
Analysis of global energy consumption and CO₂ emission trends using Machine Learning and interactive dashboard.

## ML Model
- Algorithm: Linear Regression
- R² Score: 0.9270 (92.7% accuracy)
- Prediction: 2024–2035

## Files
- ML_Energy_Prediction.ipynb → ML Model (Google Colab)
- energy_dashboard_final.html → Interactive Dashboard
- energy_data_cleaned.csv → Dataset
- prediction.csv → ML Predictions
